# Rentora Privacy Policy

This repository contains the official legal and policy documents for **Rentora**, a broker-free room rental platform built to simplify room discovery, property listing, and communication between tenants and owners.

## Documents Included
- **Privacy Policy**
- **Terms & Conditions / Terms of Service**
- **Refund / Cancellation Policy** *(if added in future)*
- **Contact / Support information**

## About Rentora
Rentora is a room rental mobile application designed to help users find and list rental properties with a smooth and transparent experience. The platform may include features such as:
- User registration and authentication
- Tenant and owner profiles
- Property listings with images and details
- Location-based property discovery
- Inquiry and communication features
- Password reset and account support

## Purpose of This Repository
This repository is maintained to publicly host Rentora’s legal pages and policy documents so they can be easily accessed from:
- Mobile application settings/help pages
- Authentication and onboarding screens
- App store privacy policy links
- Public support and compliance references
